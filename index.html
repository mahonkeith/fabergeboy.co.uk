Lists

Ordered Lists

Ordered lists can be created that have numbering. They, like unorder lists, use the "li" or "list item" tag. For example a list of largest cities in the UK by population could read:

  1. London
  2. Birmingham
  3. Manchester
  4. Liverpool
  5. Leeds

Unordered lists

Unordered lists provide the same but without an order of preference or suggested weighting. For example, a shopping list:

  • Milk
  • Flour
  • Eggs
  • Lemons

Breaks

Breaks are used to create line breaks between text


This should not be used for styling. Styling should be done in CSS. You can also use a "hr/" tag for a horizontal rule
